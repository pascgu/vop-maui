# vop-maui
A MAUI application with HybridWebView for development on Android, Windows, Linux, iOS, Mac, and web-app, using low-level features specific to each platform.

# structure example
```
vop-maui/
├── src/
│   ├── frontend/
│   │   └── ReactFlow/
│   ├── backend/
│   │   └── C#/
│   └── main.py
├── docs/
│   └── README.md
├── .gitignore
└── LICENSE
```
